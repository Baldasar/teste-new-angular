# 🌿 Projeto Angular 17 - Login com NgRx, Formulários com Formly e API Externa

Este projeto é uma aplicação moderna desenvolvida com **Angular 17**, utilizando **Standalone Components**. Ele exemplifica boas práticas com **NgRx** para gerenciamento de estado, **Formly** para construção dinâmica de formulários, e integração com uma **API externa** para demonstração de dados.

## 🚀 Tecnologias Utilizadas

- [Angular 17](https://angular.io/)
- [NgRx Store & Effects](https://ngrx.io/)
- [Formly](https://formly.dev/)
- [RxJS](https://rxjs.dev/)
- [SCSS](https://sass-lang.com/)
- [TypeScript](https://www.typescriptlang.org/)
